What is the scope of XTM Alliance?

XTM Alliance will not develop standards or overlap existing standards development efforts. Instead, the project will keep track of the developments in standard development organizations and implement these standards as they are developed and approved by aviation regulators. This will ensure that the XTM Alliance system remains up-to-date with the latest in digital aviation technology and regulations.

Therefore, the scope of the project will focus solely on providing a reference implementation for aviation standards for safe integration of new vehicles and on data exchange and flight safety / security.

How is XTM Alliance related to InterUSS and UTM Adapter projects?

We hope to build a close relationship with both these Linux Foundation projects to maintain compatibility with the APIs developed in. e.g. the Argon Server passes the verification tests developed in InterUSS. In the future we can explore a more formal link e.g. MoU or some other mechanism can be explored.

How can my institution get involved with XTM Alliance?

There are several ways to connect with the XTM Alliance community: Share a feature idea or a bug in the Github project. Join the XTM Alliance Slack workspace. You can email XTM Alliance maintainers if you have a specific questions about the project.

What are the benefits of joining XTM Alliance?

XTM Alliance provides state of the tools for UTM deployment and security, at the moment we don't have a paid membership option but that will come in the coming months. Till that time when you join the Alliance you get access to the testflight instance for one year. The test instance will give you an opportunity to explore and get familiar with the software In addition you can have a say in the direction and features of the tool set.
